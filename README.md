# Sales Performance Dashboard | Power BI

## Project Overview

This project focuses on transforming raw sales data into actionable business insights through interactive visualizations and storytelling using Power BI.

The objective was not only to create charts but also to communicate a compelling business narrative that helps stakeholders understand sales trends, customer behavior, profitability, and business opportunities.

---

## Tools Used

- Power BI
- Microsoft Excel / CSV
- GitHub

---

## Dataset

Dataset: Superstore Sales Dataset

The dataset contains information related to:

- Orders
- Customers
- Products
- Categories
- Sales
- Profit
- Regions

---

## Dashboard Objectives

- Analyze overall sales performance
- Identify high-performing categories
- Compare regional performance
- Evaluate profitability trends
- Highlight key business opportunities

---

## Key Visualizations

### Sales by Category
Analyzes revenue contribution across product categories.

### Profit Analysis
Identifies profitable and loss-making segments.

### Regional Performance
Compares sales performance across geographical regions.

### Sales Trend Analysis
Tracks sales growth over time.

### Top Products
Highlights products contributing the most revenue.

---

## Key Insights

- Technology generated the highest sales.
- Certain product categories contributed high revenue but lower profitability.
- Regional performance varied significantly across markets.
- A small number of products contributed disproportionately to total sales.
